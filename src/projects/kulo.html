Kulo

Kulo is a streamlined service to book Black & yellow cabs in Mumbai.

Key Features

  • Search for Black & Yellow cabs near current location.
  • Book a cab to anywhere in Mumbai.
  • Live tracking the ride
  • Online payment using Paytm
  • Ride history with an option to email invoice

Application Overview

Sign-Up/Login

  • Signing up or loging in only requires an active phone number to receive OTP as an sms
  • OTP is automatically read from SMS
  • OTP can also be manually entered incase the permission to read SMS is denied or SIM is not present in the current phone.

Booking a Ride

  1. Pickup Location - Current Location is automatically selected, can be changed within 2.5 Km radius
  2. Drop Location - Places, saved or otherwise, can be searched and selected.
  3. Cab Type - Either Taxi or Cool Cab can be selected.
  4. Payment Mode - Payment mode can be switched between Cash or Paytm and a default mode can be set in Settings.

Tracking the Ride

  • Tracking - Once the booking is successful, driver details along with live location is displayed.
  • Cancel - There is an option to cancel before the ride has started.
  • Share - Ride details can be shared as text to other applications ( WhatsApp, Messaging etc.)
  • Emergency - Once the ride has started, there is an Emergency button to call the police.
  • OTP - While ending the ride, driver enters the fare amount for cashless rides and user OTP is used as a confirmation.
